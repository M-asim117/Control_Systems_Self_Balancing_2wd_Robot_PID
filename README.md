# Control_Systems_Self_Balancing_2wd_Robot_PID
 The Challenge
The objective was to achieve stable autonomous balancing by continuously reading sensor data and dynamically controlling motor movement through PID feedback mechanisms.
🔧 Hardware Components

ESP32-S3 Development Module
MPU6050 (Gyroscope & Accelerometer)
L298N Motor Driver
LM2596 DC-DC Buck Converter
7.4V LiPo Battery
2WD Robot Chassis

💻 Software Implementation

Language: C
Platform: Arduino IDE on ESP32-S3
Key algorithms: PID control, sensor data processing, real-time motor control

⚙️ The Key Challenge
PID tuning was the most demanding phase. Small variations in proportional, integral, and derivative parameters significantly affected balancing performance. This required extensive iteration, testing, and optimization to achieve stable autonomous control.
✅ Result
After multiple refinement cycles, the robot successfully achieved stable self-balancing in real-time—a major milestone in understanding control systems and embedded development.
